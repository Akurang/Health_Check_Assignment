# Health_Check_Assignment
A simple health check service that monitors 3 endpoints  and exposes metrics via /health using python as the programming language and Flask as the web framework.

Packages to install 
-Flask

Run the code
- In your terminal, type python health_check.py
- Click the link generated  http://127.0.0.1:5000
- Add /health to the end of it ( http://127.0.0.1:5000/health) to get the JSON response for each endpoint.
